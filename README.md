# 新年祝福网站 🎉

一个精美、现代的静态网站，用于发送新年祝福给朋友和家人。

## 功能特性 ✨

- 🎨 **精美现代的设计**：渐变色彩方案，玻璃态卡片效果
- ⏰ **实时倒计时**：精确到秒的新年倒计时
- 💌 **个性化祝福**：可自定义朋友姓名和祝福语
- 🎆 **动画效果**：五彩纸屑、烟花和闪烁星星
- 🌓 **主题切换**：深色/浅色模式
- 📱 **响应式设计**：适配所有设备尺寸
- 🔊 **音效**：新年庆祝音乐（可选）

## 快速开始 🚀

1. **访问网站**：启用GitHub Pages后，访问 `https://emoeem.github.io/hello-new-years/`
2. **本地运行**：直接打开 `index.html` 文件

## 启用GitHub Pages 🌐

要将此网站部署到GitHub Pages，请按以下步骤操作：

1. 访问GitHub仓库：https://github.com/emoeem/hello-new-years
2. 点击 **Settings**（设置）选项卡
3. 在左侧菜单中找到 **Pages**（页面）
4. 在 **Source**（源）部分，选择 **main** 分支
5. 点击 **Save**（保存）
6. 等待几分钟，您的网站将在 `https://emoeem.github.io/hello-new-years/` 上线

## 使用说明 📝

1. 打开网站后，您会看到实时新年倒计时
2. 阅读预设的新年祝福语
3. 在"发送你的祝福"部分，输入朋友的名字和个性化祝福语
4. 点击"生成祝福卡片"创建个性化祝福
5. 使用"复制祝福语"或"分享祝福"按钮分享给朋友
6. 点击右下角的"切换主题"按钮切换深色/浅色模式
